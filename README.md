# Tech Media Website

Modern, responsive website built with React, Vite, and Tailwind CSS.

## Features

- Responsive design
- Modern UI/UX
- Fast performance with Vite
- Tailwind CSS for styling
- Single Page Application with smooth scrolling

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Technology Stack

- React 18
- Vite
- Tailwind CSS
- React Router DOM
